# Full Stack Todo Application

## Tech Stack

Backend:
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

Frontend:
- React
- Axios
- Hooks

## Features

- Create Todo
- View Todos
- Update Todo
- Delete Todo
- Mark Complete / Incomplete
- Filter by completed (Backend supported)

## How to Run

### Backend
cd todo
./mvnw spring-boot:run

Runs on:
http://localhost:8080

### Frontend
cd frontend
npm install
npm start

Runs on:
http://localhost:3000
